# Static Website Deployment on AWS EC2

## Project Overview
This project demonstrates how to launch an EC2 instance on AWS and host a static website using Apache Web Server.

## Technologies Used
- AWS EC2
- Linux (Ubuntu/Amazon Linux)
- Apache Web Server
- SSH

## Steps Performed
1. Created EC2 instance
2. Configured Security Group (Port 80 enabled)
3. Installed Apache Web Server
4. Deployed index.html
5. Accessed website using Public IP

## Output
Website successfully hosted on AWS EC2 instance.
